How to Use
  • Use the Character Slot bar to manage multiple characters — create, duplicate, or delete builds.
  • Pick your class & race on the right, then browse/add skills using the Skills tab.
  • Enter BP budget directly — level is calculated as floor(BP / 10) for body point formulas. Body Points = 10 base + class multiplier × level + Fortify.
  • BP budget tracks automatically; remaining BP turns red if over-budget.
  • Magic section handles Spell Slots, Formal Magic, and Cantrips for primary & secondary schools.
  • GM tab tracks work shift hours and exports a sign-off PDF.
  • All characters auto-save to your browser. Use Export PDF for a printable character sheet.
  • Use Save Link or Copy Link to encode the current build into the URL for sharing.
⚠️ Prereq checker flags text-based prerequisites (e.g. "Weapon Proficiency +2") but does not enforce every rule automatically. Always verify complex chains manually.
